Hey, how are you?

Today we are going to study how to define classes in Ruby, because like everything in it is an object, it's about time we know how to set up ours:

[code lang=ruby]
class Book
end
[/code]

Everytime we create one class it's good that we write the method initialize, which is always called when a new instance of our class is made. The well known constructor:

[code lang=ruby]
class Book
def initialize
end
end
[/code]

Ruby doesn't support method override, so only the last one is considered. As an alternative, we can use default arguments (a tip is to use them at the end of our list, because Ruby has a strange behavior otherwise, see here)

[code lang=ruby]
class Book
def initialize(author, name, code = '1')
puts "author: #{author}, name: #{name}, code: #{code}"
end
end
[/code]

This way, if we instanciate an object without the code value, it's all good:

[code lang=ruby]
class Book
def initialize(author, name, code = '1')
puts "author: #{author}, name: #{name}, code: #{code}"
end
end

book1 = Book.new('Cecilia Assis', 'Ruby 101')
book2 = Book.new('Julia Tolezano', 'Ta todo mundo mal', '12345')
[/code]

Outcome:

author: Cecilia Assis, name: Ruby 101, code: 1
author: Julia Tolezano, name: Ta todo mundo mal, code: 12345

So, as we can see, because of what we've defined for the initialize method, the infos about the Book are shown.

With the example above we already know how to declare an object and defined the first thing that is going to happen with it, right?
All we need to do is: give it a name, write the initialize method and instanciate it.
