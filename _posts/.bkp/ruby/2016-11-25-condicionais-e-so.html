Pois bem! Hoje vim falar de condicionais, os famosos if e elses e como o Ruby os trata. Let's go!

If

Bem, todo programador sabe que em algum momento uma decisão deverá ser tomada sua aplicação. Se algo acontecer o que eu faço? No Ruby:

[code lang="ruby"]
if algumaCoisaAconteceu
puts 'O que eu faço?' # haha, saida: O que eu faço?
end
[/code]

Perceba que o Ruby requisita a keyword end no final, para informar o fim do if, lembrando as chaves {}, utilizadas na Linguagem C, por exemplo.

If Else

O se-senão no Ruby também pede o comando end no final, mas somente para o else.

[code lang="ruby"]
if number > 0
puts 'Positivo' # saida: Positivo
else
puts 'Negativo ou Zero' # saida: Negativo ou Zero
end
[/code]

If Elsif Else

Como apresentado no título, aqui a linguagem mostra mais um diferencial. Enquanto C e Java, por exemplo usam a terminologia else if o Ruby utiliza elsif.

[code lang="ruby"]
if number > 0
puts 'Positivo' # saida: Positivo
elsif number == 0
puts 'Zero' # saida: Zero
else
puts 'Negativo' # saida: Negativo
end
[/code]

É possível perceber com os códigos já vistos e será nos próximos que o Ruby não pede parênteses (), mas também não impede que eles sejam colocados, logo, cabe a você a decisão.

[code lang="ruby"]
if (number > 0)
puts 'Positivo' # saida: Positivo
end

# igual

if number > 0
puts 'Positivo' # saida: Positivo
end
[/code]
