Hey, como vai!?

Hoje vim falar do famigerado vetor, vulgo Array. O Ruby traz várias coisas bem legais com e para ele, e vamos ver algumas delas nesta série de posts.

Para declarar um array podemos fazer da seguinte forma:

[code lang="ruby"]
ary = [1, 2, 3, 4]
aro = []
[/code]

O Ruby permite array com valores de vários tipos:

[code lang="ruby"]
ary = ['a', 1, 'casinha', 2.1, true]
#=> ['a', 1, 'casinha', 2.1, true]
[/code]

É uma boa prática que os separadores dos elementos, as vírgulas, sejam acompanhadas por espaços. Melhor compreensão do código.

Quando queremos declarar um array com um tamanho inicial pré-definido, podemos usar explicitamente sua classe:

[code lang="ruby"]
ary = Array.new(3)
#=> [nil, nil, nil]
[/code]

Se quisermos algum valor inicial:

[code lang="ruby"]
ary = Array.new(3, 'ruby')
#=> ['ruby', 'ruby', 'ruby']
[/code]

Acessando índices únicos

Existem algumas formas de acesso a valores em Ruby. Uma delas é bem conhecida pelos programadores de C/C++ e JS:

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary[0]
puts ary[1]
puts ary[2]
[/code]

Saída:

ruby
oi
gente

Como pudemos ver, o Ruby também faz a contagem de índices iniciando do zero (0). Logo, a string 'ruby' tem índice 0, e assim por diante.

Outra maneira de acesso é:

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary.at(0)
#=> ruby
[/code]

Acessando ranges (intervalos)

Outra característica sensacional do Ruby é que ele permite o acesso índices por meio de intervalos. Enquanto em outras linguagens precisamos fazer um laço para isso, aqui temos outras formas.

O uso de pontos finais é uma delas. Dependendo do número de pontos finais no comando, diferentes informações são passadas ao interpretador:

  1. Dois pontinhos -> (..): Informa ao interpretador que queremos os valores do índice x ao y, incluindo o y.

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary[0..2]
[/code]

Saída:

ruby
oi
gente

  1. Três pontinhos -> (...): Informa ao interpretador que queremos os valores do índice x ao y, não incluindo o y.

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary[0...2]
[/code]

Saída:

ruby
oi

As vírgulas também podem ser usadas, e entre elas temos primeiro o índice de onde queremos começar e segundo quantos números após ele. Logo se temos:

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary[0, 2]
[/code]

Saída:

ruby
oi

Dizemos ao interpretador, que a partir do índice 0 queremos 2 valores, isto é o valor no índice e mais um. Vamos a outro exemplo:

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary[0, 1] #=> ruby
[/code]

BÔNUS TIME !!

Quando se trata do acesso a índices no vetor, o Ruby também traz uma funcionalidade muito legal: O índice negativo. Este conta de trás para frente os elementos do array, começando do 1 e mostra o valor no local desejado.

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary[-1]
#=> gente
[/code]

Podemos usar dessa ferramenta para o acesso em ranges idem. Com o mesmo array anterior, fazemos:

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary[0..-2]
[/code]

Saída:

ruby
oi

Usando a vírgula:

[code lang="ruby"]
ary = ['ruby', 'oi', 'gente']
puts ary[-2, 2]
[/code]

Saída:

oi
gente
