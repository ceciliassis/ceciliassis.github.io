Represented by , the remarkeble Spaceship Operator works the same way, that his endings: like a comparison opertor, three-way comparison, being a key element in sorting methods.

The little spaceship (got it?) counts with three types of comparision, always done from left to right, and can return three values as well:

  • -1: if the left value is less than the right;
  • 0: if the values are equal;
  • 1: if the left value is bigger than the right.

[code lang=ruby]
puts 1 <=> 2
#=> -1
puts 1 <=> 1
#=> 0
puts 2 <=> 1
#=> 1
[/code]
