Hey, como vai?!

Hoje vamos estudar como definir classes no Ruby e como tudo em Ruby é um objeto, já não será sem tempo que aprendêssemos como definir os nossos:

[code lang=ruby]
class Livro
end
[/code]

Toda vez que criamos uma classe é bom que criemos também o método initialize, chamado sempre que uma nova instância de nossa classe é criada, o famoso construtor.

[code lang=ruby]
class Livro
def initialize
end
end
[/code]

O Ruby não suporta sobrecarga de métodos em classes, e somente o último estabelecido é considerado. Como uma alternativa, usamos argumentos com valores padrão, uma dica é usá-los no final da listagem de argumentos, porque o Ruby tem um comportamento estranho caso contrário, veja aqui:

[code lang=ruby]
class Livro
def initialize(autor, nome, isbn = '1')
puts "Autor: #{autor}, nome: #{nome}, ISBN: #{isbn}"
end
end
[/code]

Dessa forma, se instanciarmos um objeto sem o valor do ISBN, tudo estará bem:

[code lang=ruby]
class Livro
def initialize(autor, nome, isbn = '1')
puts "Autor: #{autor}, nome: #{nome}, ISBN: #{isbn}"
end
end

livro1 = Livro.new 'Cecilia Assis', 'Ruby 101'
livro2 = Livro.new 'Julia Tolezano', 'Ta todo mundo mal', '12345'
[/code]

Saída:

Autor: Cecilia Assis, nome: Ruby 101, ISBN: 1
Autor: Julia Tolezano, nome: Ta todo mundo mal, ISBN: 12345

Bem, com o exemplo acima já sabemos como declarar um objeto, certo? [nomeDaClasse].new . O texto acima é projetado, pois foi o que definimos para o método initialize.
