Hey, tá joia?

Gente do céu, o Ruby é mágico.

Cá estava eu, brincando um pouco, aprendendo um pouco e me deparei com algo super legal: subtração e adição de arrays; mas, o que é isso, você deve se perguntar ... eu te mostro!

[code lang=ruby]
a = [1, 2]
b = [3, 4]
c = a + b

p c
[/code]

A saída desse código será:

[1, 2, 3, 4]

Isso mesmo! A concatenação dos arrays!!!!!

A subtração também pode ser operada:

[code lang=ruby]
a = [1, 2]
b = [3, 4]
c = a + b

p c-a
[/code]

Saída:

[3, 4]

Ai, o Ruby SEMPRE me surpreende (:

Existem tantas aplicações para isso, você conhece alguma? Me diga nos comentários!
