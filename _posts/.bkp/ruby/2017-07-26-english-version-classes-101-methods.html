Hey, how are you?

Like said on the first post of this serie, today we are going to know how we add new methods to our class, and so much more.

Inside the method initialize we can have anything and today we are going to use it to "initialize" our variables.

[code lang=ruby]
class Book
def initialize(author, name, isbn)
@author = author
@name = name
@isbn = isbn
puts "Printing the book \"#{@name}\""
end
end
[/code]

We use the @ when we declare a instance variable, meaning the it only belongs to the object.

Well, now that we have the variables we can define a method that shows their content, like any other.

[code lang=ruby]
def about
puts "Printing the book \"#{@name}\""
end
[/code]

The whole code:

[code lang=ruby]
class Book
def initialize(author, name, isbn)
@author = author
@name = name
@isbn = isbn
puts "Printing the book \"#{@name}\""
