Hey, como vai?!

Hoje voltamos a falar dos vetores e seus muitos métodos.

Tamanho

Para sabermos a quantidade de elemento no array, usamos:

[code lang="ruby"]
ary = [1, 2, 3, 4]
puts ary.count
# ou
puts ary.length
[/code]

Para ambos a saída será: 4.

Adicionar elementos

O Ruby traz algumas abordagens ... A primeira é conhecida pelos programadores de JS.

[code lang="ruby"]
ary = [1, 2, 3, 4]
ary.push(5)
puts ary #=> 0, 1, 2, 3, 4, 5
[/code]

Como pudemos ver, o item foi adicionado no final do vetor, a próxima forma também insere no fim deste.

[code lang="ruby"]
ary = [1, 2, 3, 4]
ary < 0, 1, 2, 3, 4, 5
[/code]

O Ruby permite que adicionemos itens em qualquer parte do vetor, com a função insert , que recebe enquanto parâmetros a posição e o ou os elementos a ser inseridos.

[code lang="ruby"]
ary = [1, 2, 3, 4]
ary.insert 2, 'oi'
puts ary #=> 1, 2, "oi", 3, 4
[/code]

[code lang="ruby"]
ary = [1, 2, 3, 4]
ary.insert 2, 'oi', 'como'
puts ary #=> 1, 2, "oi", "como", 3, 4
[/code]

A última abordagem insere itens no início do array.

[code lang="ruby"]
ary = [1, 2, 3, 4]
ary.unshift(0)
puts ary #=> 0, 1, 2, 3, 4
[/code]

É importante ter em mente que todos aceitam a inserção de mais de um elemento por vez, e para tal basta listar mais argumentos. Faça um teste!

Removendo elementos

Alguns métodos que aqui serão abordados, também já são conhecidos. Algo que já pudemos perceber pelas nosso aprendizado constante é que o Ruby utiliza de conceitos comuns a outras linguagens.

Aqui, enquanto o método pop retira o último elemento, shift retira o primeiro.

[code lang="ruby"]
ary = [1, 2, 3, 4]
ary.pop() #=> 4
ary.shift() #=> 1
puts ary #=> 2, 3
[/code]

A remoção por index também é permitida:

[code lang="ruby"]
ary = [1, 2, 3, 4]
ary.delete_at(1) #=> 2
puts ary #=> [1, 3, 4]
[/code]

Por fim, por hoje, a remoção de um elemento específico igualmente é possível por meio da função delete. Mas fique atento: todas as ocorrências do item serão removidas.

[code lang="ruby"]
ary = [1, 2, 2, 3, 4]
ary.delete(2) #=> 2
puts ary #=> [1, 3, 4]
[/code]
