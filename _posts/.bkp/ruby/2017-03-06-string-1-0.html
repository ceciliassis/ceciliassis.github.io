Hey, como vai?!

Vamos falar de Strings? Como usual, usarei da documentação para trazer métodos que considero importantes, dentre os já estudados por mim.

Já sabemos que para declarar uma string, basta declarar:

[code lang="ruby"]
hello = 'hello'
[/code]

E para visualizar, basta imprimir:

[code lang="ruby"]
hello = 'hello'
puts hello
#=> hello
[/code]

Aqui as Strings também são vetores, e como tais podemos acessá-los:

[code lang="ruby"]
hello = 'hello'
puts hello[0..2]
#=> hel
[/code]

STRING * 3

Bem, antes de falarmos do mais comum, falemos de algo inusitado, a multiplicação de Strings.

Feita de forma bem simples, basta adicionarmos um asterisco (*) e o número de vezes que queremos que tal string seja reproduzida:

[code lang="ruby"]
hello = 'hello'
puts hello * 2
#=> hello
#=> hello
[/code]

Legal, né?

CONCATENAÇÃO

Enquanto opções de concatenação temos:

[code lang="ruby"]
hello = 'hello'
hello < hello Rubist

hello += ', how are you'
puts hello
#=> hello Rubist, how are you

hello.concat('?')
puts hello
#=> hello Rubist, how are you?
[/code]

E O TAMANHO?

Para tal, temos as mesmas keywords que usamos com os vetores: size e lenght, todas informando o mesmo dado. O count por outro lado, significa um método que conta quantidade de chars que dentro da string correspondem ao parâmetro dado, veja: String#count.

[code lang="ruby"]
hello = 'Hello'
puts hello.size, hello.length
#=> 5
#=> 5
[/code]

CASES

Se quisermos a primeira letra maiúscula:

[code lang="ruby"]
puts hello.capitalize
#=> Hello
[/code]

E todas as letras?

[code lang="ruby"]
puts hello.upcase
#=> HELLO
[/code]

Minúsculas?

[code lang="ruby"]
hello = 'HELLO'
puts hello.downcase
#=> hello
[/code]

Podemos até inverter a caixa das letras?

[code lang="ruby"]
hello = 'Hello'
puts hello.swapcase
#=> hELLO
[/code]

REVERSE

Por fim, se quisermos a string ao contrário ...

[code lang="ruby"]
hello = 'Hello'
puts hello.reverse
#=> olleH
[/code]
