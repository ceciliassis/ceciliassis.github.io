Print vs Puts

Ambos os códigos servem para impressão de algo na tela. O comando puts insere um new line ( \n ) no fim de cada chamada, enquanto o print não o faz.

[code lang="ruby"]
puts 'Hello, World!'
#=> Hello, World!
print 'Hello, World!'
print 'Hello, World!'
#=> Hello, World!Hello, World!
[/code]

Variáveis

O Ruby não é uma linguagem fortemente tipada como o Java e por isso é de fácil entendimento. Muito semelhante ao JavaScript uma vez que a variável é declarada essa pode assumir qualquer valor/tipo.

É convencional que os nomes associados às variáveis sejam minúsculos e separados por underline ou underscore ( _ ).

[code lang="ruby"]
o_boleano = true
puts o_boleano
#=> true

a_string = 'Hello, World'
puts a_string
#=> Hello, World

o_numero = 5
puts o_numero
#=> 5
o_numero = 5.5
puts o_numero
#=> 5.5
[/code]

Cálculos Aritméticos

Soma, divisão e afins funcionam no Ruby como já conhecemos, porém a exponenciação é diferente, logo:

[code lang="ruby"]
puts 6 + 1
#=> 7
puts 6 - 1
#=> 5
puts 15 % 6
#=> 3
puts 10**2
#=> 100
[/code]

Como visto, a exponenciação é feita por meio de dois asteriscos ( * ), no exemplo 10x10 ou 10^2.

Comentários

Todo bom programador gosta de comentar seus códigos, seja para ler e entender depois, servir de lembrete para algo pendente ou documentá-los. Como visto acima, os comentários em linha única são marcado por #, o famoso jogo-da-velha. Para comentarmos em múltiplas linhas, fazemos:

[code lang="ruby"]
=begin
Estou no blog me, Ruby Girl!
Nao precisa do # .
=end

# Somente uma linha ...
[/code]

Curiosidades

Para saber a qual classe pertence certo objeto no Ruby, fazemos:

[code lang="ruby"]
o_numero = 5
puts o_numero.class
#=> FixNum
o_numero = 5.5
puts o_numero.class
#=> Float
[/code]

Nesse código é possível perceber algo interessante: O tão conhecido int em Ruby é FixNum. 

Quanto as strings, o Ruby aceita aspas simples ( ' ' ) ou duplas ( " " ), preferindo simples para uso corriqueiro e duplas para caracteres especiais (aspas simples na frase, por exemplo) e concatenação de strings.
