Hey, como vai?!

Como sabemos, tudo em Ruby é um objeto, por isso métodos são declarados e não funções já que essas não incidem sobre um objeto.

Para declararmos um método usamos:

[code lang="ruby"]
def hello_world
puts 'Bem vindo ao Ruby!'
end
[/code]

A chamada do já declarado método é feita da seguinte forma:

[code lang="ruby"]
hello_world
[/code]

Saída:

Bem vindo ao Ruby!

Com esse exemplo, talvez você pense: Mas esse método não pertence a classe alguma.

Porém lhe digo que sim, ele faz parte da classe Method. Para perceber isso é necessário algumas coisa mais avançadas, com o Ruby instalado em seu PC:

  1. Abra o terminal
  2. Digite irb
  3. Declare o método que quiser, com o nome que quiser
  4. Digite Method.instance_methods

Pois bem, esse passos mostrarão um vetor contendo junto a outros métodos, o novo feito. Esses pertencem a classe Method e a ela somente já que são métodos de instância.

Eu fiz tudo isso com o exemplo trazido, o resultado foi:

Então, como pôde ver, o método hello_world está lá.

Usei também um notação que traz somente os 26 primeiros elementos do array, assunto do post: Vetores e Vetores, parte 1.

Uma forma mais simples seria digitar method(:hello_world) no irb. Resultado:

#

Dizendo que o método hello_world pertence a classe Object, mãe de todas as outras.

É importante salientar que se digitarmos Object.instance_methods no irb, o novo método declarado também aparecerá dentre os demais.

O Ruby traz os métodos de uma classe enquanto Symbols (símbolos), um conceito amplamente usado na linguagem e que será abordado posteriormente.

E é somente isso que precisamos para declaração e uso dos métodos. O código inteiro fica assim:

[code lang="ruby"]
def hello_world
puts 'Bem vindo ao Ruby!'
end

hello_world
[/code]

Métodos parametrizados

Estamos muito acostumados a uma série de parâmetros em nossos métodos, no Ruby é assim:

[code lang="ruby"]
def hello_world(frase)
puts frase
end

hello_world 'Bem vindo ao Ruby!'
# ou
hello_world('Bem vindo ao Ruby!')
[/code]

Saída:

Bem vindo ao Ruby!

Bem vindo ao Ruby!

Como já dito, no Ruby podemos ou não usar parênteses na chamada de métodos parametrizados.  Também é bom saber que para separação dos parâmetros são usadas vírgulas (,).

Métodos com valores default (padrão)

Existe a opção de valores padrão para parâmetros:

[code lang="ruby"]
def hello_world(frase = 'Bem vindo ao Ruby')
puts frase
end

hello_world
hello_world 'Ruby é muito legal!'
[/code]

Saída:

Bem vindo ao Ruby!

Ruby é muito legal!

Métodos com retorno

Em Ruby a última linha do método é dada enquanto o seu retorno, porém o comando return também pode ser usado.

[code lang="ruby"]
def hello_world
'Bem vindo ao Ruby'
end

frase = hello_world
puts frase
[/code]

Saída:

Bem vindo ao Ruby

Usando a keyword return, teríamos:

[code lang="ruby"]
def hello_world
return 'Bem vindo ao Ruby'
end

frase = hello_world
puts frase
[/code]

Saída:

Bem vindo ao Ruby

O mesmo vale para condicionais dentro do método.

[code lang="ruby"]
def hello_world(option)
if(option == 1)
'Bem vindo'
else
'Até mais'
end
end

puts hello_world 0
puts hello_world 1
[/code]

Saída:

Bem vindo ao Ruby
